### Usage explanation ###

1. Place following files (if different than default) in location of "MonsterSetBase Converter.exe"
	+ MonsterList.xml
	+ MonsterSetBase.txt
	+ MonsterSetBase_CS.txt
	
 NOTE: DO NOT replace MapList.xml to the one from server files 
 
2. Run MonsterSetBase Converter.exe -> a "MonsterSetBase Conversions" folder will be created with newly converted IGC_MonsterSpawn.xml file
